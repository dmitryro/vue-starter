# VueJS starter with Webpack 4, nginx and Docker

### Steps to get started:

### Clone the repo
``` 
   git clone git@github.com:dmitryro/vue-starter.git
```

### Build the Docker containers
``` 
   docker-compose build
```

### Start the Docker containers
```
   docker-compose up
```

### Open in browser:
```    
   http://0.0.0.0:8080
```
